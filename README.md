# My Portfolio Site

This is a personal portfolio website built to showcase my projects. It features a blog-style layout where each project is presented as a post with descriptions, tech details, and screen recordings. The site is designed as a single-page application for a clean user experience, with sections for about me, projects, and contact.

## Features

- **Single-Page Layout**: Hero/about section, project grid with expandable tiles, and contact form.
- **Project Showcases**: Currently featuring 3-6 projects (e.g., Hadith RAG Chat, Chess AI Game), with plans to add more.
- **Content Management**: Powered by Sanity.io for easy blog post management.
- **Responsive Design**: Mobile-friendly using Tailwind CSS.
- **Tech Integrations**: Embedded videos from YouTube/Vimeo, dynamic data fetching from Sanity.

## Tech Stack

- **Frontend**: Next.js (with TypeScript), Tailwind CSS, shadcn/ui
- **CMS**: Sanity.io (headless CMS for blog posts)
- **Deployment**: Vercel
- **Other**: JavaScript for logic, react-player for video embeds

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn
- A Sanity.io account (free tier)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-portfolio.git
   cd my-portfolio
   ```
